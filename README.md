# Arctic Game (base site)

A clean starter site for your school game hub. **No games included** — add them later.

## Files

- `index.html`: layout + tabs (Home / Music / Changelog)
- `styles.css`: clean modern styling
- `app.js`: tab switching + local music player + playlist

## Add music

1. Create a folder: `assets/music/`
2. Put audio files there (MP3/OGG)
3. Edit `TRACKS` in `app.js` to match your filenames.

## iTunes 30-second previews (free)

The Music tab also supports **iTunes Search API** previews (30 seconds).

- Type a search in Music → Search.
- Click a result to play the preview.
- Use the **Apple** link on a result to open it on Apple/iTunes.

## Run it

Just open `index.html` in your browser (double click).
